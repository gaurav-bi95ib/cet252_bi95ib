# Game Review Client Prototype

This client was built using Vanilla HTML, CSS, and JS to maintain simplicity, blazing speed, and no complex compilation overhead. It uses absolute best practices in design with dark mode, aesthetic glassmorphism effects, modern web typography, and responsive grid layouts.

## Running the Client
1. Be sure the API is already running (`npm start` in the `../api` folder).
2. Simply open `index.html` in any Modern Web Browser (Chrome, Firefox, Safari). You can simply double-click it. 
   *(Alternatively, serve via VS Code live server or `npx serve .`)*

## Functional Testing (TestCafe)
This client utilizes TestCafe for functional frontend testing as requested.

1. Install dependencies: `npm install`
2. Run test suite: `npm test`
   - *This will open Chrome dynamically and assert UI behaviors.*
